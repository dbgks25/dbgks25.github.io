## 안녕하세요!

이곳은 제가 직접 작성한 **논문**과 수행한 **연구과제**를 정리해 두는
개인 아카이브입니다. 데이터 분석과 환경·공간 연구를 주로 다룹니다.

### 관심 분야

- **데이터 분석 & 시각화**: Python, QGIS를 활용한 데이터 분석 및 시각화
- **공간·환경 연구**: 위성영상, 지도 데이터를 활용한 환경 분석
- **공간통계**: 공간 자기상관, 회귀 기반 추정

### 기술 스택

#### 데이터 분석
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- R (ggplot2, dplyr, spdep)
- Jupyter Notebook, QGIS
